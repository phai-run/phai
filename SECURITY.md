# Security Policy

## Supported Versions

Only the latest minor release receives security fixes. Older versions are not supported.

| Version | Supported |
|---------|-----------|
| latest  | ✅ |
| < latest | ❌ |

## Reporting a Vulnerability

If you discover a security vulnerability, please report it privately via
[GitHub Security Advisories](https://github.com/phai-run/phai/security/advisories/new).

Do **not** open a public issue.

You can expect:
- An initial response within 7 days
- A fix or mitigation within 30 days of confirmation
- Credit in the release notes (unless you request anonymity)
