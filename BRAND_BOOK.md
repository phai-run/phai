## 1. Identidade

### Nome

```
phai
```

Pronúncia: **fai** (como "fly" sem o L, ou "fácil" sem o "cil").

### Símbolo

```
φ
```

A letra grega phi (U+03C6). O número de ouro: 1.6180339887...

### Três camadas de significado

| Camada | Significado |
|--------|-------------|
| **φ (phi)** | Proporção áurea. Equilíbrio matemático. |
| **Finance + AI** | Agente financeiro inteligente. |
| **phai.run** | Executa. Roda. Ação. |

### O domínio é o manifesto

```
phai.run
```

`.run` não é só um TLD — é o verbo do produto. `phai run` = o comando que inicia tudo.

```bash
$ phai run daily-pulse
$ phai run sync
$ phai run forecast
```

## 3. Tom de Voz

### Como falamos

```
✅ "phai run sync — 247 transações."
✅ "Saldo projetado: R$ 3.247,00."
✅ "3 assinaturas recorrentes. Revisar?"

❌ "Olá! Vamos ver suas financinhas?"
❌ "Parabéns! Você economizou R$ 12,50!"
❌ "Seu score financeiro é 7.8."
```

### Regras

1. **Nunca infantilize.** O usuário é dev.
2. **Vá direto ao ponto.** Informação > conversa.
3. **Terminal-first.** Tudo funciona em 80 colunas.
4. **Dados, não opiniões.** "Gastos subiram 23%." Não "Você está gastando muito!"
